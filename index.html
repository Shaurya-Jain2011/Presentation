Mission Solar Tuk-Tuk

A Scalable Hybrid Solution for Sustainable Urban Mobility

The Urban Challenge

In India, tuk-tuks (auto-rickshaws) are the go-to transport for millions, zipping through crowded streets where larger vehicles can't. Despite their convenience, they suffer from major inefficiencies: they use polluting fuel, contribute to urban emissions, and waste energy.

The Problem

High Pollution, Fuel Cost, and Inefficiency.

The Objective

A solar-hybrid system for minimum cost and virtually infinite runtime.

Prototype: "Coco"

Our proof-of-concept is "Coco," a miniature four-wheeler demonstrating the hybrid solar-battery system.

16 cm Total Length
7 cm Width / Height
2 cm Wheel Diameter

System Architecture

A modular system designed for efficiency. Power flows from the solar panel to charge the battery, which simultaneously powers the motors (via the L298N driver) and the control logic (via a 5V regulator).

Scalable Speed Control

The Arduino control code is built for scaling. Speed for the entire system is managed by a single, easily adjustable variable. 

int speedLevel = 180; // (0-255)

void moveForward() {
  analogWrite(ENA, speedLevel);
  analogWrite(ENB, speedLevel);
  // ... direction pins ...
}

This variable can be replaced with a potentiometer, allowing for dynamic, real-time speed adjustment in a full-scale model.

The Formulas for Future Development

Linear Speed (v)

v = π * D * (RPM / 60)

  • D: Wheel Diameter (m)
  • RPM: Motor Rotations / Minute

Required Torque (T)

T = F * r

  • F: Force at Wheel Rim (N)
  • r: Wheel Radius (m)

Motor Power (P)

P = T * ω

  • T: Torque (N·m)
  • ω: Angular Speed (rad/s)

Electrical Current (I)

I = P / V

  • P: Power (Watts)
  • V: Motor Supply Voltage (V)
